# 💰 PayCheck

Understand the true cost of your loan with precise clarity.

## Features

- Calculate monthly EMI instantly
- See total interest you will pay
- See total repayment amount
- Clean minimal design

## Built With

- HTML
- CSS
- JavaScript

## How to Use

1. Enter your loan amount
2. Enter annual interest rate
3. Enter loan tenure in years
4. See your EMI and repayment breakdown instantly
